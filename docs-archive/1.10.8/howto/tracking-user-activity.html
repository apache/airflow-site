Tracking User Activity¶

You can configure Airflow to route anonymous data to Google Analytics, Segment, or Metarouter.

Edit airflow.cfg and set the analytics block to have a tool and id:

[webserver]
# Send anonymous user activity to Google Analytics, Segment, or Metarouter
analytics_tool = google_analytics # valid options: google_analytics, segment, metarouter
analytics_id = XXXXXXXXXXX

Note

You can see view injected tracker html within Airflow’s source code at airflow/www_rbac/templates/appbuilder/baselayout.html. The related global variables are set in airflow/www_rbac/templates/app.py.
